About Flask Projects

This dashboard helps you manage your local Flask development environments. Here's what you need to know:

  • Project Path: The full directory path where your Flask project is located.
  • Entry File: The main Python file that runs your Flask application (often app.py, main.py, run.py).
  • Virtual Environment: If your project uses a virtual environment named "venv" at the project path, it will be used automatically.
  • Port: The port number your Flask application will listen on (default is 5000).
  • Multiple Projects: You can run multiple Flask applications simultaneously by assigning different ports to each project.
